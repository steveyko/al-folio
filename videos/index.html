Rushmore

Rushmore is a system capable of displaying confidential images on a mobile device even when the OS is compromised. Rushmore makes use of ARM TrustZone to enable such a capability but goes well beyond just using TrustZone.

PDF

FLUID

FLUID enables users to seamlessly move UI elements such as a chat box or a video playback from one device to a different device. FLUID support this for unmodified apps, such as Netflix and Google Meet. For more information, check out the website at http://cps.kaist.ac.kr/?page=research/fluid/contents.html.

PDF

X-Droid

X-Droid empowers developers to quickly test out new features of their apps by integrating functions from other existing apps. X-Droid makes this possible by combining a technique called programming-by-demonstration (PBD) with a system that can run an existing app without showing any visual output. For more information and other videos, check out the website at https://nmsl.kaist.ac.kr/projects/xdroid/.

PDF

BlueMountain

BlueMountain imagines a new mobile ecosystem where users can customize the behavior of their storage by installing storage plugins---regular apps that control how read/write calls should behave. For more information, check out the website at http://bluemountain.cse.buffalo.edu.

PDF

Reptor

Reptor introduces a new bytecode instrumentation technique called API virtualization, which enables developers to change the behavior of an Android API call as they wish without modifying the Android OS. For more information and videos, check out the website at http://reptor.cse.buffalo.edu.
